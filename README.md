# Программная защита от копирования (учебная реализация)

Файлы:

- `installer.py` — генератор лицензии, привязанной к компьютеру
- `protected_app.py` — защищаемая программа (проверяет `license.key`)
- `license.key` — генерируется `installer.py` и размещается рядом с приложением

Как использовать

1. На компьютере, где нужно установить программу, запустите установщик:

```powershell
python installer.py --out-dir .
```

Команда создаст файл `license.key` в текущей директории и выведет идентификатор устройства и ключ.

2. Запустите защищаемое приложение (в той же папке, где `license.key`):

```powershell
python protected_app.py
```

При совпадении ключа приложение запустится и откроет простой калькулятор.
При несовпадении будет выведено сообщение об ошибке лицензии и приложение завершит работу.

Сборка в .exe (PyInstaller)

1. Установите PyInstaller:

```powershell
pip install pyinstaller
```

2. Соберите `protected_app.py` в exe (пример):

```powershell
pyinstaller --onefile protected_app.py
```

3. Скопируйте сгенерированный `license.key` рядом с `protected_app.exe` на целевой машине.

Проверка сценариев

- Успешный запуск: запуск `protected_app.py` на компьютере, где был сгенерирован `license.key` — приложение запускается.
- Неуспешный запуск: копирование `protected_app.py` и `protected_app.exe` на другой компьютер без переноса `license.key` или с `license.key`, сгенерированным для другого устройства — при запуске вы увидите ошибку лицензии.

Ограничения

Реализация учебная: не защищает от реверс-инжиниринга, лицензия хранится в открытом виде, методы идентификации простые.
